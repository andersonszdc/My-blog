> ## Introdução

Esta plataforma web foi desenvolvida para a venda de assinaturas e infoprodutos. 

<blockquote className="with-icons">
    ## Tecnologias utilizadas
    <TechIcons techs={['nextjs', 'styledComponents', 'firebase']}/>
</blockquote>

Este projeto foi desenvolvido utilizando Create Next App por ter um ótimo SEO, SSR e o roteamento nativo do Next.js. Escolhi utilizar Styled-components para a estilização por já estar familiarizado.

As outras principais dependências que eu utilizei foram:

- Firebase Authentication para cadastro de usuários.
- Firebase Firestore como banco de dados dos usuários e produtos.
- Firebase Cloud Functions para disparo de funções automáticas como a integração do Firebase com o Stripe.
- Youtube e Vimeo para hospedar vídeos.
- Stripe como gateway de pagamento.

> ## Desafios

Por ser primeira vez no universo Firebase, tive bastantes consultas ao stackoverflow e documentação hehe, inclusive a doc do Firebase é bem clara e objetiva (recomendo).

> ## Aprendizado

Este projeto fez eu estudar bastante o Firebase, aprendi na prática sobre computação em nuvem. Consegui explorar mais sobre clean code nete projeto, por já não ser o primeiro.
