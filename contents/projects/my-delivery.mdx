> ## Introdução

Este projeto web foi desenvolvido para delivery's que querem ter um web app de vendas e informações sobre a loja. 

<blockquote className="with-icons">
    ## Tecnologias utilizadas
    <TechIcons techs={['nextjs', 'styledComponents', 'mongodb']}/>
</blockquote>

Este projeto foi desenvolvido utilizando Create Next App por ter um ótimo SEO, SSR e o roteamento nativo do Next.js. Escolhi utilizar Styled-components para a estilização por já estar familiarizado.

As outras principais dependências que eu utilizei foram:

- mongoDB como banco de dados para registro de pedidos e cardápios.
- Mongoose para modelagem de objetos para o mongoDB.
- Stripe como gateway de pagamento.

> ## Desafios

Este projeto explorou bastante a utilização da Context API para fornecer globalmente informações como o pedido do cliente e informações para entrega.

Aproveitei a API routes do Next.js para criar esta aplicacação full-stack, as api's para o POST do pedido e GET, POST, DELETE e PUT do cardápio foram criadas na mesma aplicação do Front-end.

> ## Aprendizado

Aprendi muito com este projeto, por conta de ser o primeiro após eu estudar o banco de dados MongoDB. A implementação do checkout transparente foi de grande aprendizado também.
