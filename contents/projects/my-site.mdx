> ## Introdução

Este projeto web foi desenvolvido para apresentar meus projetos como freelancer e registrar snippets interessantes e features das tecnologias que eu utilizo. 

<blockquote className="with-icons">
    ## Tecnologias utilizadas
    <TechIcons techs={['nextjs', 'styledComponents', 'firebase']}/>
</blockquote>

Este projeto foi desenvolvido utilizando Create Next App por ter um ótimo SEO, SSR e o roteamento nativo do Next.js. Escolhi utilizar Styled-components para a estilização por já estar familiarizado.

As outras principais dependências que eu utilizei foram:

- firebase para hospedar imagens, cloud functions e também utilizo a integração dele com o Analytics 
- nodemailer para envio de emails automáticos
- handlebars para adicionar variáveis no corpo do email
- mdx-bundler para compilar os arquivos MDX com os componentes React.

> ## Desafios

O desenvolvimento de um motor de leitura de MDX (como este que você está lendo rs) foi bem desafiador, com a biblioteca mdx-bundler e algumas funções nativas do Node.js consigo escrever em MDX, que é bem mais simples do que HTML, e adicionar componentes React, e então compilá-lo automaticamente.

Há também um outro motor de leitura HTML onde eu envio e-mails automáticos para quem me envia mensagem na aba Contatos, este foi desenvolvido antes do leitor de MDX, por isso eu ainda não o refatorei.

> ## Aprendizado

Aprendi muito com este projeto, por conta de ser o primeiro após eu estudar a teoria do Next.js e Typescript. Este projeto continua sendo mantido para adicionar novas features.
